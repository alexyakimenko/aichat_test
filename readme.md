# AIChat Test

## Installation process

### Clone repository
```
git clone <link>
cd aichat_test/
```
### Run project (backend)
```
cd backend
npm run dev
```
### Run project (frontend)
```
cd fontend
npm run dev
```
### .env template
```dotenv
GROQ_API_KEY= # your api key
PORT= # backend server port (optional)
```
